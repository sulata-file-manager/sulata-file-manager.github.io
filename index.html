Sulata File Manager

Free, lightweight file manager in PHP,
ideal for installation on VPS machines

Features

- Copy
- Paste
- Delete
- Rename
- Zip*
- Unzip*

- Download
- Upload
- Change Permission
- Create File
- Create Folder
- Edit File

Requirement

- PHP 5.3 or above
- shell_exec() function must be enabled in PHP and zip from terminal functionality installed on OS to zip/unzip*
